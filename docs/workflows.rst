Workflows
=========

Teacher Workflow
----------------
1. Upload reading materials in Documents.
2. Generate a paper using AI, edit questions, and publish.
3. Assign the paper to a class or student.
4. Review analytics and class performance.

Student Workflow
----------------
1. Open assignment from Home.
2. Take the paper (timed). Submit answers.
3. Return to reading paper view with grading status.
4. Review results after grading.
