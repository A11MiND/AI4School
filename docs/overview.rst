Overview
========

AI4School is an educational platform that helps teachers manage classes, upload reading materials,
create AI-generated exams, assign papers, and monitor student performance. Students can access
assigned materials, complete papers, and review their progress.

Key Features
------------
- Role-based access (teacher, student, admin)
- Document management with per-class visibility
- AI-assisted paper generation and editing
- Timed student paper UI with objective + open answer types
- Analytics dashboards for teachers and reports for students
